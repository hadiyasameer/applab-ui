# UI Developer Assessment

Project Overview

This is a responsive web page project based on the provided prototype. 
It includes all major features from the prototype, along with subtle improvements for UX, accessibility and performance.

How to Run Locally
1. Download or clone the project to your local machine.
2. Navigate to the project folder.
3. Open index.html in your browser.
 
 This project runs using HTML, CSS and vanilla JavaScript Only. Front-end frameworks such as React, Angular and Vue were intentionally avoided, as requested in the assessment instructions.

 Approach
 - Mobile-first design: Most Sections were built Starting from smaller screen and then  scaled up for desktop
 - Semantic HTML : Used appropriate HTML tags to improve accessibility and readability
 - CSS Structure : Tried to organize styles into Base, Layout,Components and utilities for maintainability
 - Interactions & animations : Implemented Hover states,mobile menu toggle, next-button actions,dropdown header , and scroll triggered for smooth experience.
 - UX improvements : Made Subtle spacing adjustments, improved hover feedback ,and accessible navigation where possible.

Trade-offs & Known Limitations
- Some animations may slightly differ from prototype due to timing adjustments for smoother  performance
- A few  sections are not fully styled due to time constraints.
